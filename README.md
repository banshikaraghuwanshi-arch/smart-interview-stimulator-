# Smart Interview Simulator

## 📌 Overview

Smart Interview Simulator is a full-stack web application that helps users practice technical interview questions in an interactive environment. It allows candidates to answer questions, save responses, and improve their interview preparation skills.

## ✨ Features

* 🎯 Practice interview questions interactively
* 📝 Submit and store answers
* 📂 Backend API for managing interview data
* ⚡ Fast and responsive React frontend
* 🗄️ MongoDB database integration
* 🔄 Real-time communication between frontend and backend

## 🚀 Installation

### Clone the repository

git clone https://github.com/banshikaraghuwanshi-arch/smart-interview-stimulator-.git

### Frontend Setup

cd smart-interview-frontend
npm install
npm run dev

### Backend Setup

cd ../smart-interview-backend
npm install
npm start

## ⚙️ Environment Variables

Create a `.env` file inside the backend folder and add the required environment variables:

MONGO_URI=your_mongodb_connection_string

PORT=5000
